# 密码管理器

一个安全的密码管理工具，帮助您存储和管理所有密码。

## 功能特性

- 🔐 **主密码保护** - 使用主密码加密所有数据
- 🔒 **AES-256 加密** - 采用 Web Crypto API 进行端到端加密
- 🎲 **密码生成器** - 生成强密码，支持自定义规则
- 📁 **分类管理** - 按类别组织密码（网站、应用、邮箱等）
- ⭐ **收藏夹** - 快速访问常用密码
- 🔍 **搜索功能** - 快速查找密码
- 📋 **一键复制** - 快速复制用户名和密码
- 👁️ **密码显示/隐藏** - 安全查看密码
- 💾 **本地存储** - 所有数据存储在本地 IndexedDB

## 安全说明

- 所有密码使用 AES-GCM 加密算法加密
- 主密码通过 PBKDF2 派生密钥（100,000 次迭代）
- 数据完全存储在本地，不会上传到任何服务器
- **重要：主密码无法找回，请务必牢记**

## 使用方法

1. 首次使用时设置主密码（至少 8 个字符）
2. 使用主密码解锁密码管理器
3. 添加、编辑或删除密码条目
4. 使用密码生成器创建强密码
5. 使用搜索和分类功能快速查找密码

## 开发

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建
pnpm build

# 打包插件
pnpm package
```
